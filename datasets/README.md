# Downloaded Datasets

This directory contains datasets for the research project.

## Dataset 1: AI-Scientist ICLR Bench
- **Source**: SakanaAI/AI-Scientist repository (review_iclr_bench)
- **Location**: datasets/AI-Scientist-ICLR/
- **Content**:
  - `ratings_subset.tsv`: ICLR paper ratings.
  - `iclr_parsed/`: Parsed ICLR papers (likely text).
  - `llm_reviews/`: Reviews generated by LLMs.
- **Task**: Automated Peer Review Evaluation.

## Dataset 2: ReviewerGPT NeurIPS Checklist
- **Source**: ReviewerGPT repository (NeurIPS_checklist/data)
- **Location**: datasets/ReviewerGPT-NeurIPS/
- **Content**: JSON files containing checklist questions and paper data.
- **Task**: Error detection and checklist verification.
